Move Zeros

Given an array, move all zeros to the end of array, and keep other non-zero elements in relative order. [0, 1, 0, 3, 12] -> [1, 3, 12, 0, 0]

Solution:

Two pointers + one pass shift non-zero values as far forward as possible. and fill remaining spaces with zeros.

Use two pointers, one points to the next position to be replaced, another points to the current iterating element. when current element is not zero, set the value to another replaced element. If it's zero, do nothing. After one pointer reaches the end, set all the element starting from replaced index to zero.

Time Complexity: O(n)

https://leetcode.com/problems/move-zeroes/

Sort Colors

Given an array with n objects colored red, white or blue, sort them so that objects of the same color are adjacent, with the colors in the order red, white and blue.

Solution:

The idea is to swap all the 0's to the left, all the 2's to the right, and all 1's in the middle. use two pointers to record next 0 position, and next 2 position.

Time Complexity: O(n)

https://leetcode.com/problems/sort-colors/
